This is the End solution of the ASP.NET MVC 4 and Entity Framework Migrations Hands-On Lab.

Follow these steps to install the **NuGet** package dependencies before running the solution.

1. Open the **NuGet** **Package Manager Console**. To do this, select **Tools | Library Package Manager | Package Manager Console**.

2. In the **Package Manager Console,** type **Install-Package NuGetPowerTools**.

3. After installing the package, type **Enable-PackageRestore**.

4. Build the solution. The **NuGet** dependencies will be downloaded and installed automatically.
